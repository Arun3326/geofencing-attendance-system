# Geofencing-Based Attendance Management System

This project is a Flask-based web application designed to manage employee attendance using geofencing.

## Features
- Role-based login (Admin & Employee)
- Location-based attendance marking
- Attendance history tracking
- Dashboard for monitoring

## Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite

## Use Case
Prevents proxy attendance and improves workforce tracking using location validation.
